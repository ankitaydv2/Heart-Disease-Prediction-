# Heart Disease Prediction using Machine Learning

This project predicts whether a person has heart disease or not using machine learning classification models.

## Models Used
- Logistic Regression
- Decision Tree Classifier

## Accuracy
- Logistic Regression: 88.52%
- Decision Tree: 85.24%

## Libraries Used
- pandas
- numpy
- scikit-learn
- matplotlib

## Workflow
1. Load dataset
2. Split data into training and testing sets
3. Train ML models
4. Predict results
5. Compare accuracy
